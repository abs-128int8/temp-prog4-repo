## gitでブランチを作る
ブランチとは枝のこと．開発において違う系統で開発したい場合はブランチを切る．
- 今のブランチを見る
```
git branch
```

- ブランチを作る
```
git branch <BranchName>
```

- ブランチを移動
```
git checkout <BranchName>
```
